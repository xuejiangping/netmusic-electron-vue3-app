- #### 2023 年 8 月

这是项目第二版截图，又学习了一年多，并且借鉴了[trtst/Vue3.x 网易云音乐\_PC 音乐播放器](https://gitee.com/trtst/vue3-music)， 决定用 electron+vue3 重写这个项目。虽然还是有不少瑕疵，但已经可以像一个真正的桌面应用程序一样运行，可以有多个窗口，有桌面级别歌词，有系统托盘控制。。。

想做的部分其实还蛮多的，但由于各种各样原因，这可能是最终形态了
